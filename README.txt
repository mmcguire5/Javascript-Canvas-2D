>index.html creates a single rectangle using HTML and javascript canvas 2D
-----------------------------------------------------------------------------------------------------------------------------------------------------

>MultiRect.html creates a canvas of 10,000 rectangles (100x100) by using nested for loops.
-----------------------------------------------------------------------------------------------------------------------------------------------------

>MultiRectThousand.html creates a canvas of 1,000,000 rectangles (1000x1000) by using nested for loops. However, displaying this canvas becomes a challenge due to height/width 	restrictions set by browsers.This issue is solved by reducing the size of each individual rectangle

---------------------------------------------------------------------------------------------------------------------------------------------------------

>JSON_JS.html handles creating a canvas that will display rectangles that are specified in a JSON file
>rectangles.json is the JSON file that contains a dynamic array which holds parameters for rectangles to be displayed on the canvas
>myScript.js parses the JSON file, sets the attributes of the canvas and displays rectangles on the canvas depending on the parameters held in the JSON file
