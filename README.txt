>MultiRect.html creates a canvas of 10,000 rectangles (100x100) by using nested for loops.
>MultiRect.html creates a canvas of 1,000,000 rectangles (1000x1000) by using nested for loops. However, displaying this canvas becomes a challenge due to height/width restrictions set by browsers.
